🐍 Python

Le langage des données — plus simple que vous ne pensez.

Vos premières analyses avec pandas, sur de vrais fichiers CSV. Quatre lignes de code suffisent pour commencer.

import pandas as pd

df = pd.read_csv("transferts_diaspora.csv")
haiti = df[df.code_iso3 == "HTI"]
print(haiti.part_pib_pct.max())  # 22.45 (record 2020)

Outils & sources de données

Chaque outil avec son lien officiel, ses étapes d'installation et une première pratique guidée — plus les meilleures sources de données gratuites du monde.

Tous les liens pointent vers les sites officiels. Certains pourront devenir des liens partenaires (affiliés) — cela ne change jamais le prix pour vous, et nous ne recommandons que ce que nous utilisons.

Gratuit

Power BI Desktop

L'outil de tableaux de bord le plus demandé sur le marché de l'emploi — et celui que nous utilisons dans nos tutoriels.

Installation & première pratique
  1. Windows uniquement. Installez via le Microsoft Store (recherchez « Power BI Desktop ») — mises à jour automatiques.
  2. Ouvrez → « Obtenir les données » → Texte/CSV.
  3. Première pratique : chargez notre dataset gratuit des transferts et créez un histogramme pays × part_pib_pct.
Site officiel Tutoriel Atmart
Gratuit

Tableau Public

L'alternative à Power BI, gratuite dans sa version Public — vos visualisations sont publiées en ligne (parfait pour un portfolio).

Installation & première pratique
  1. Créez un compte gratuit sur public.tableau.com → téléchargez Tableau Public (Windows/Mac).
  2. Connectez un fichier texte (CSV).
  3. Première pratique : refaites la courbe « Haïti, % du PIB 2000–2024 » du tutoriel Excel — et publiez-la : c'est votre première pièce de portfolio.
Site officiel
Gratuit

Python

Le langage n°1 de la donnée et de l'IA. Avec pandas, 4 lignes suffisent pour analyser un CSV.

Installation & première pratique
  1. Téléchargez sur python.org → ⚠ cochez « Add Python to PATH » à l'installation.
  2. Installez pandas : ouvrez un terminal → pip install pandas matplotlib.
  3. Petite connexion ? Utilisez Google Colab (colab.google) : Python dans le navigateur, zéro installation.
  4. Première pratique : import pandas as pd; df = pd.read_csv("transferts_diaspora_2000_2025.csv"); print(df.head())
Site officiel Colab (sans installation)
Gratuit

R + RStudio

La référence des statisticiens et des chercheurs — très utilisé pour les enquêtes (LAPOP, recensements) et l'économétrie.

Installation & première pratique
  1. Installez R depuis cran.r-project.org, puis RStudio Desktop depuis posit.co (dans cet ordre).
  2. Première pratique : donnees <- read.csv("transferts_diaspora_2000_2025.csv"); summary(donnees)
  3. Puis : hist(donnees$part_pib_pct) — votre premier histogramme en une ligne.
R (CRAN) RStudio
Gratuit

Java (JDK)

Le langage des grandes applications d'entreprise et d'Android — une base solide pour l'ingénierie logicielle.

Installation & première pratique
  1. Téléchargez le JDK LTS gratuit chez Adoptium (Eclipse Temurin) — adoptium.net.
  2. Vérifiez : terminal → java -version.
  3. Première pratique : créez Bonjou.java avec System.out.println("Bonjou Ayiti!"); → javac Bonjou.java → java Bonjou.
Adoptium (officiel)
Payant / étudiant

EViews

Le logiciel d'économétrie des banques centrales et ministères — séries temporelles, prévisions macro.

Installation & première pratique
  1. Licence payante — version EViews Student Lite gratuite sur eviews.com (inscription e-mail).
  2. Première pratique : importez la série des transferts d'Haïti et tracez-la (Quick → Graph) ; testez une moyenne mobile.
  3. Alternative 100 % gratuite pour l'économétrie : gretl (gretl.sourceforge.net).
Site officiel
Gratuit (cloud)

SAS OnDemand

SAS est exigé dans les banques, assurances et la pharma. La version académique cloud est gratuite — rien à installer.

Installation & première pratique
  1. Créez un compte gratuit sur SAS OnDemand for Academics (welcome.oda.sas.com).
  2. Tout se passe dans le navigateur (SAS Studio).
  3. Première pratique : importez le CSV des transferts → proc means data=work.import; var part_pib_pct; run;
SAS OnDemand
Payant

Stata

Le standard des économistes du développement, des ONG et des enquêtes ménages (Banque mondiale, USAID).

Installation & première pratique
  1. Licence payante sur stata.com (tarifs étudiants réduits ; souvent fournie par les universités et ONG).
  2. Première pratique : import delimited transferts_diaspora_2000_2025.csv → summarize part_pib_pct if code_iso3=="HTI".
  3. Pas de licence ? R fait tout ce que Stata fait — commencez par R.
Site officiel
Gratuit

MySQL + Workbench

La base de données la plus utilisée du web. Le SQL est la 2e compétence la plus demandée chez les analystes, juste après Excel.

Installation & première pratique
  1. Téléchargez MySQL Community Server + MySQL Workbench sur dev.mysql.com/downloads.
  2. À l'installation, notez bien votre mot de passe root.
  3. Première pratique : importez le CSV des transferts (Table Data Import Wizard) puis : SELECT pays, MAX(part_pib_pct) FROM transferts GROUP BY pays;
Site officiel
Gratuit

GitHub + Git

Votre CV technique vivant : chaque projet publié sur GitHub est une preuve de compétence pour les recruteurs et clients freelance.

Installation & première pratique
  1. Créez un compte gratuit sur github.com.
  2. Installez Git : git-scm.com (Windows/Mac/Linux).
  3. Première pratique : créez un repo « mon-premier-projet-data », ajoutez votre analyse Excel du tutoriel Atmart et un README qui explique vos conclusions.
GitHub Git

VS Code

L'éditeur de code universel — Python, R, SQL, web — avec des milliers d'extensions gratuites.

Installation & première pratique

  2. Installez les extensions Python et Jupyter (onglet Extensions).

Gratuit (cloud)

Google Colab

Python + IA dans le navigateur, zéro installation — la meilleure option si votre ordinateur est modeste ou votre connexion limitée.

Installation & première pratique
  1. Allez sur colab.google avec un compte Google — c'est tout.
  2. Première pratique : Nouveau notebook → glissez le CSV des transferts dans les fichiers → pd.read_csv(...) → graphique en 3 lignes.
  3. Bonus : GPU gratuit pour vos premiers modèles d'IA.
